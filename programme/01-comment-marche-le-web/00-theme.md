# Thème 01 — Comment marche le web

> **Le but :** construire le *modèle mental* du web avant d'écrire la moindre ligne de
> code. Quand on comprend qui parle à qui, on ne code plus à l'aveugle.

## Pourquoi ce thème en premier

Beaucoup de débutants apprennent à écrire du HTML sans jamais comprendre ce qui se
passe quand on tape une adresse et qu'une page apparaît. Résultat : ils sont perdus
dès qu'un bug sort de l'écran (« pourquoi ça marche chez moi mais pas en ligne ? »).
Ce thème pose les fondations pour ne plus jamais être dans le flou.

## Ce que l'apprenant saura faire à la fin

- Expliquer avec ses mots le trajet d'une page, de la barre d'adresse à l'écran.
- Distinguer client et serveur, front et back, statique et dynamique.
- Lire une URL et un code de statut HTTP (200, 404, 500…).
- Ouvrir les devtools et comprendre ce qu'est le DOM.
- Dire pourquoi un site doit être en HTTPS.
