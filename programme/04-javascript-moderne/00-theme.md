# Thème 04 — JavaScript moderne

> **Le but :** apprendre le **langage du web**, celui qui rend les pages interactives —
> et qui est le socle direct de **React**. On vise un JavaScript **moderne** (ES6+),
> lisible et propre, des bases jusqu'aux appels d'API.

## Pourquoi ce thème est décisif

JavaScript est le seul langage qui tourne nativement dans le navigateur. Le maîtriser
**avant** React est non négociable : React n'est « que » du JavaScript bien organisé.
Beaucoup de difficultés en React viennent en réalité de bases JS fragiles (tableaux,
fonctions, asynchrone). On consolide donc tout ça ici.

## Ce que l'apprenant saura faire à la fin

- Écrire des fonctions, manipuler **tableaux et objets** (`map`/`filter`/`reduce`).
- Utiliser la syntaxe **moderne** (arrow, destructuring, modules `import`/`export`).
- **Manipuler le DOM** et réagir aux **événements** (clic, saisie, soumission).
- Faire des appels **asynchrones** à une **API REST** avec `fetch` et `async/await`.
- **Déboguer** son code avec les devtools et comprendre l'intérêt d'ESLint/Prettier.
