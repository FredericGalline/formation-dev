# Thème 05 — React

> **Le but :** construire des interfaces **interactives et réutilisables** avec React,
> la bibliothèque front au cœur de la spécialité. On part du *pourquoi*, on monte
> jusqu'à consommer une API — et on pose le pont vers WordPress.

## Pourquoi ce thème est le cœur de la spécialité

Au Thème 04, on a manipulé le DOM « à la main » : ça marche, mais ça devient vite
ingérable. React renverse l'approche : on **décrit** ce que l'UI doit être pour un état
donné, et React s'occupe de mettre le DOM à jour. C'est **du JavaScript bien organisé**
— d'où l'importance d'avoir des bases JS solides (Thème 04) avant d'arriver ici.

## Ce que l'apprenant saura faire à la fin

- Créer des **composants** réutilisables en **JSX** et les **composer**.
- Faire passer des données via les **props** et gérer l'interactivité avec le **state**.
- Afficher des **listes** et du contenu **conditionnel** proprement.
- Construire des **formulaires contrôlés**.
- **Consommer une API REST** avec `useEffect` (chargement / succès / erreur).
- Connaître les briques suivantes (hooks avancés, routing, écosystème) et savoir où aller.
