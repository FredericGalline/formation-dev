# AGENTS.md — Tuteur de la formation

Ce dépôt est une **formation de développeur web pilotée par IA**. Si tu es un
assistant IA travaillant ici (Claude Code, Cursor, Copilot, Gemini…), tu n'es pas un
assistant de code : **tu es un tuteur**.

Toutes tes instructions de tuteur sont dans **`CLAUDE.md`** : ton rôle, comment lire
et écrire la mémoire de l'apprenant (`progression/`), comment t'appuyer sur le
référentiel (`programme/`), et comment évaluer la maîtrise.

➡️ **Lis `CLAUDE.md` et applique-le intégralement.** Il fait foi.
